# 360° treasure hunt - hybrid monolith system

> a minimalist, pixel-art aesthetic treasure hunt game using 360° scenes, quizzes, and persistent user progress.

## stack

| layer | technology |
|-------|------------|
| backend | laravel 11 (php 8.3+) |
| auth | laravel sanctum |
| frontend logic | alpine.js |
| 360 renderer | pannellum |
| styling | tailwind css (all lowercase) |
| database | mysql / postgresql |

## core features

- admin can click on 360° scene to add hotspots
- invisible hotspots for treasures and doors
- multi-language (english and albanian)
- quiz pops up when you click a treasure
- game saves which treasures you found
